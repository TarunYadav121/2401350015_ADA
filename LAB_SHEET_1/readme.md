# Lab Sheet 1 - Sorting Algorithms Analysis

## Implemented Algorithms
- Bubble Sort (Optimized)
- Selection Sort
- Insertion Sort

## Analysis Method
Step-count method used to count number of comparisons.

## Cases Analyzed
- Best Case (Sorted Input)
- Worst Case (Reverse Sorted Input)
- Average Case (Random Input)

## Input Sizes
10, 20, 30, 40

## Observations
- Selection Sort always performs O(n²) comparisons.
- Bubble Sort performs O(n) in best case due to optimization.
- Insertion Sort performs efficiently for sorted input.

## How to Run
1. Compile:
   javac LabSheet1Sorting.java
2. Run:
   java LabSheet1Sorting
